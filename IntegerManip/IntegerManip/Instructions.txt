Source Control Activity #5 - Integer Manipulation

As a team, write a project to prompt the user for up to 20 integers.  The user will be allowed to choose from the following menu options until they decide to quit:

	• Display the whole list of input
	• Display every even number
	• Display every odd number
	• Display every 5th number
	• Display the sum of the integers
	• Display every number divisible by 3
	• Quit

Team - Be sure to use proper source control techniques to manage pull requests

Include additional features time permitting.
